# Parvez Portfolio React

Public React/Vite version of the portfolio website.

This package is for Vercel/public sharing. It does not include the visible editor toolbar. Use the separate private editable HTML file for changes.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The profile image is embedded in the app, so it does not need a separate image file.
